# ORM-backend

ORM-Backend is a command-line
application that allows a user
to update an existing database
with products, categories, and
tags.


## Installation

"dotenv": "^16.0.1",
"express": "^4.18.1",
"mysql2": "^2.3.3",
"nodemon": "^2.0.19",
"sequelize": "^6.21.4"
